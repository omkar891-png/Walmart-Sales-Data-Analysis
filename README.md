# Walmart Sales Data Analysis

## 📌 Project Overview
This project focuses on analyzing Walmart retail sales data to uncover insights about sales performance, product categories, customer trends, and regional performance. The analysis was performed using **SQL for data querying and transformation** and **Power BI for interactive data visualization**.

The goal of this project is to demonstrate how raw sales data can be transformed into meaningful insights that support **data-driven business decisions**.

---

## 🎯 Objectives
- Analyze Walmart sales data to identify key sales trends.
- Understand the performance of different product categories.
- Evaluate the impact of discounts on sales.
- Compare sales performance across regions.
- Create interactive dashboards for easy data interpretation.

---

## 🛠️ Tools & Technologies
- **SQL** – Data querying and data analysis  
- **Power BI** – Data visualization and dashboard creation  
- **CSV Dataset** – Source of Walmart sales data  
- **Database Management System** – For storing and managing data  

---

## 🔄 Project Workflow
1. Collected Walmart sales dataset in **CSV format**.
2. Imported the dataset into a **SQL database**.
3. Performed **data cleaning and transformation using SQL queries**.
4. Connected the SQL database to **Power BI**.
5. Built **interactive dashboards** to visualize sales insights.

---

## 📊 Key Insights
- Identified top-performing product categories.
- Analyzed monthly sales trends and revenue patterns.
- Compared sales performance across different regions.
- Observed the relationship between **discounts and sales volume**.

---

## 📈 Dashboard Features
The Power BI dashboard includes:
- Sales performance overview
- Regional sales comparison
- Product category analysis
- Monthly revenue trends
- Discount impact on sales
